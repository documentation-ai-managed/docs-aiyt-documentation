<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added tree view toggle for nested JSON navigation
- Introduced one-click copy to clipboard for selected paths
- Support for JSON Schema validation with custom schemas

## Improvements

- Enhanced beautify options with custom indentation levels
- Faster parsing for files `>10MB`

## Bug Fixes

- Fixed collapse/expand state persistence in large objects
- Resolved validation errors on escaped Unicode characters

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Dark mode support with automatic theme detection
- Export beautified JSON as HTML or Markdown tables
- Search and highlight functionality across the entire JSON

## Breaking Changes

- Updated validation engine; migrate custom rules if needed

## Bug Fixes

- Corrected rendering issues with circular references
- Improved error messages for malformed JSON inputs

<CodeGroup tabs="Before,After">
```json
// Before: Manual search needed
{
  "users": [
    {"id": 1, "name": "Alice"},
    {"id": 2, "name": "Bob"}
  ]
}
```
```json
// After: Search "Alice" highlights instantly
{
  "users": [
    {"id": 1, "name": "Alice"},
    {"id": 2, "name": "Bob"}
  ]
}
```
</CodeGroup>

</Update>
