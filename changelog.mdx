## Overview

Use this changelog to record how JSONViewer.AI evolves over time. Each entry should capture what changed in a given release so users can understand new capabilities, fixes, and any breaking behavior.

This page is a **template**: it does not list real historical releases. Maintainers should replace the placeholder content below with actual release notes as new versions are published.

<Callout kind="info">

JSONViewer.AI runs fully in the browser as a free JSON viewer, validator, and beautifier. When you update behavior, performance, or UI, document it here so users know what to expect.

</Callout>

## How to use this changelog

Use one `<Update>` block per release. For each release, try to capture:

- **Version identifier**: a semantic version, hash, or short label.
- **Release date**: the day the change became publicly available.
- **Highlights**: what matters most to users (features, fixes, improvements, known issues).
- **Impact**: anything users must do differently (for example, breaking changes or changed shortcuts).
